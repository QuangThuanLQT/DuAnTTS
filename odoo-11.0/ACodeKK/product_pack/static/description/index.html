PRODUCT PACK

Product pack allows you to create the packs of products to enhance your bussiness and attract your customers. Create Combo packs on special occassions and festivals for increasing your sale.

Basic Features

  • Odoo's Product pack is an easy to use interface for managing the products in packs.
  • It is fully integrated and helps to create the packs of products.
  • This Module allows to configure the products as a collection of other products.
  • It is similar to product and have all the properties of the pack.
  • Manages the stock of the pack.
  • Have the seperate option for adding only packs for generating Sales Order
  • Have the option for adding pack only or products only or both on creating a Picking of Sales Order.
  • Have many other additional features.

Additional Features

Some key points

  • The default vaules will be used while creating the vouchers.
  • Vouchers can not be created below and above the minimum and maximum value resp.
  • Vouchers can be created for some Specific customer or for All customers
  • Same voucher can be used for the Ecommerce & Odoo Core or POS or For Both.
  • Depending upon the customer type selected we can partially use the voucher, and also specify the maximum limit of the partial use.
